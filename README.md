# CozyKV

Validates data containing key-value pairs.

## Installation

Add `:cozy_kv` to the list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:cozy_kv, <requirement>}
  ]
end
```

## Usage

For more information, see the [documentation](https://hexdocs.pm/cozy_kv).
